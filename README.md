# CovalentX | Next-Gen SaaS Workflow Automation 🤖

> The ultimate dark-themed SaaS platform for modern developers.

CovalentX is a responsive, next-generation landing page for a SaaS workflow automation platform. Built with a sleek dark theme and glassmorphism elements, it provides a beautiful, high-converting presentation of the platform's features, pricing, and social proof.

## ✨ Features

- **Responsive Design:** Fully responsive layout that looks great on mobile, tablet, and desktop devices.
- **Modern Dark UI:** Sleek, high-contrast dark theme optimized for developer-focused tools.
- **Dynamic Animations:** Scroll-triggered reveal animations, counter stats, and smooth scrolling navigation.
- **Interactive Navigation:** Sticky navigation bar with scroll progress indicator and a mobile-friendly hamburger menu.
- **Glassmorphism Elements:** Premium design aesthetics featuring frosted glass effects, glowing borders, and floating badges.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure and content.
- **CSS3:** Custom variables, Flexbox/Grid layouts, glassmorphism (`backdrop-filter`), and CSS animations.
- **Vanilla JavaScript:** DOM manipulation, scroll events, `IntersectionObserver` for animations, and smooth scrolling.

## 🚀 Getting Started

### Prerequisites

You only need a web browser to view this project. No build tools or backend servers are required.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/kanishknegi2006-oss/responsive-landing-page.git
   cd responsive-landing-page
   ```

2. **Open the application:**
   Simply open the `index.html` file in your preferred web browser:
   ```bash
   # On macOS
   open index.html
   
   # On Windows
   start index.html
   
   # On Linux
   xdg-open index.html
   ```
   Alternatively, you can use an extension like **Live Server** in VS Code for a better development experience.
